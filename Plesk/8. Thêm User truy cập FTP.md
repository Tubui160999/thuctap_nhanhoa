# Add User FTP 
- Tại Plesk Control Panel -> Chọn `Webmail & Domains` -> `FTP Access` 

![](./images/FTP.png)

- Chọn `Add an FTP Account` để tạo tài khoản FTP 

![](./images/createftp.png)

- Điền thông tin cần thiết, chọn thư mục mà User có quyền quản lý và click `OK`

![](./images/ftp1.png)

![](./images/ftpus.png)
